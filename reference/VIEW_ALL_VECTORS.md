# View all vectors specifier

A pair to use in the `data` parameter of `viewer` to expose all base
vectors with their original names. See the Julia documentation for
details.

## Usage

``` r
VIEW_ALL_VECTORS
```

## Format

A named list representing the pair ALL_VECTORS =\> "="
