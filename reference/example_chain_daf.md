# Load example chain data into a Daf object

Load example chain data into a Daf object

## Usage

``` r
example_chain_daf(name = "chain!")
```

## Arguments

- name:

  Name for the Daf object (default: "chain!")

## Value

A Daf object containing chained example cells and metacells data

## Details

This function creates a chain of example cells and metacells data. See
the Julia
[documentation](https://tanaylab.github.io/DataAxesFormats.jl/v0.1.2/example_data.html)
for details.
