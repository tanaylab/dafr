# Get set of scalar names from a Daf object

Returns the names of all scalar properties in the Daf data set.

## Usage

``` r
scalars_set(daf)
```

## Arguments

- daf:

  A Daf object

## Value

A character vector of scalar property names

## Details

This function provides the complete set of available scalar properties
that can be retrieved using
[`get_scalar()`](https://tanaylab.github.io/dafr/reference/get_scalar.md).
See the Julia
[documentation](https://tanaylab.github.io/DataAxesFormats.jl/v0.1.2/readers.html#DataAxesFormats.Readers.scalars_set)
for details.
