# Empty caches on a Daf object.

Specify at most one of `clear` or `keep`. Tier names use the short form
(`"mapped"`, `"memory"`, `"query"`) or the Julia-style capitalised form
(`"MappedData"`, `"MemoryData"`, `"QueryData"`).

## Usage

``` r
empty_cache(daf, clear = NULL, keep = NULL)
```

## Arguments

- daf:

  A `DafReader`/`DafWriter` instance.

- clear:

  Character vector of tiers to clear (default: all tiers).

- keep:

  Character vector of tiers to keep; all others are cleared.

## Value

Invisibly the input `daf`.
