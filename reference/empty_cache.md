# Empty cache in a Daf object

Empty cache in a Daf object

## Usage

``` r
empty_cache(daf, clear = NULL, keep = NULL)
```

## Arguments

- daf:

  A Daf object

- clear:

  Cache group to clear. Can be one of "MappedData", "MemoryData", or
  "QueryData".

- keep:

  Cache group to keep. Can be one of "MappedData", "MemoryData", or
  "QueryData".

## Value

The Daf object
