# View all axes specifier

A pair to use in the `axes` parameter of `viewer` to expose all base
axes with their original names. See the Julia documentation for details.

## Usage

``` r
VIEW_ALL_AXES
```

## Format

A named list representing the pair ALL_AXES =\> "="
