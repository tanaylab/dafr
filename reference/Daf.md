# Daf (Data Axes Format) S3 object

An S3 object that wraps a Julia DataAxesFormats object and provides
access to its methods.

## Usage

``` r
Daf(jl_obj)
```

## Arguments

- jl_obj:

  The Julia DafReader object to wrap

## Value

A Daf S3 object
