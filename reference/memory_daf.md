# Create a Daf object with in-memory storage

This function creates a Daf object that stores data in memory. See the
Julia
[documentation](https://tanaylab.github.io/DataAxesFormats.jl/v0.1.2/memory_format.html)
for details.

## Usage

``` r
memory_daf(name = "memory")
```

## Arguments

- name:

  The name of the Daf object (default: "memory")

## Value

A Daf object with in-memory storage
