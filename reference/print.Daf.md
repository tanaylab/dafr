# Print method for Daf objects

Prints a description of the Daf object using Julia's description
function

## Usage

``` r
# S3 method for class 'Daf'
print(x, ...)
```

## Arguments

- x:

  The Daf object to print

- ...:

  Additional arguments passed to print

## Value

The Daf object (invisibly)

## Details

See the Julia
[documentation](https://tanaylab.github.io/DataAxesFormats.jl/v0.1.2/readers.html#DataAxesFormats.Readers.description)
for details.
