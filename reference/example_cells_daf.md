# Load example cells data into a Daf object

Load example cells data into a Daf object

## Usage

``` r
example_cells_daf(name = "cells!")
```

## Arguments

- name:

  Name for the Daf object (default: "cells!")

## Value

A Daf object containing example cells data

## Details

This function loads example cells data from the DataAxesFormats.jl
package. See the Julia
[documentation](https://tanaylab.github.io/DataAxesFormats.jl/v0.1.2/example_data.html)
for details.
