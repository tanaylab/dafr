# Unescape a value from a query string

Reverses the escaping done by `escape_value`, restoring the original
value.

## Usage

``` r
unescape_value(value)
```

## Arguments

- value:

  A character string value to unescape

## Value

The unescaped value as a character string

## Details

See the Julia
[documentation](https://tanaylab.github.io/DataAxesFormats.jl/v0.1.2/queries.html#DataAxesFormats.Queries.unescape_value)
for details.
