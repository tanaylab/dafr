# View all data specifier

A list of pairs to use in the `data` parameter of `viewer` to expose all
base data (scalars, vectors, and matrices) with their original names.
See the Julia documentation for details.

## Usage

``` r
VIEW_ALL_DATA
```

## Format

A list containing VIEW_ALL_SCALARS, VIEW_ALL_VECTORS, and
VIEW_ALL_MATRICES
