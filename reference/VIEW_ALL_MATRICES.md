# View all matrices specifier

A pair to use in the `data` parameter of `viewer` to expose all base
matrices with their original names. See the Julia documentation for
details.

## Usage

``` r
VIEW_ALL_MATRICES
```

## Format

A named list representing the pair ALL_MATRICES =\> "="
