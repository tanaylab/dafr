# Merge action constants for [`concatenate()`](https://tanaylab.github.io/dafr/reference/concatenate.md).

Used as values in
[`concatenate()`](https://tanaylab.github.io/dafr/reference/concatenate.md)'s
`merge` argument: `MERGE_SKIP` skips the property (default),
`MERGE_LAST_VALUE` uses the last source containing it,
`MERGE_COLLECT_AXIS` collects values along the `dataset_axis`.

## Usage

``` r
MERGE_SKIP

MERGE_LAST_VALUE

MERGE_COLLECT_AXIS
```

## Format

An object of class `character` of length 1.

An object of class `character` of length 1.

An object of class `character` of length 1.
